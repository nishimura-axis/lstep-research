📝 まとめ

🔵 全事例で共通している要素

  • 予約形式 → 全て「承認リクエスト型」（即時確定ではない）
  • セグメント分け → LINE追加後または予約後に質問でタグ付け
  • クーポン訴求 → 何らかの特典・割引を提示
  • フォーム経由での予約 → myfm.jpまたは自社フォーム使用
  • 複数希望日時の取得 → 第2〜4希望まで取得
  • 「カウンセリングのみ可」の選択肢 → 予約ハードルを下げる設計

🔴 事例ごとの特徴的な違い

  • LP設計 → Juno Beautyのみ通常LP、他3事例はアンケート形式LP
  • 緊急性訴求 → DIOクリニックがカウントダウンタイマー + 残り枠数で最も強い
  • セグメント取得タイミング → DIOクリニックのみ予約後に取得（他は予約前）
  • 個人情報取得タイミング → DIOクリニックがフォーム最初で名前・電話番号取得
  • フォームツール → ABCクリニックのみ自社フォーム（他はmyfm.jp）
  • 複数治療促進 → ABCクリニックが「よくばりキャンペーン」で複数治療割引
